# 什么是 KageSU？

一个 Android 上基于内核的 root 方案。

::: info 基于 SukiSU-Ultra
**KageSU** 是 [SukiSU-Ultra](https://github.com/SukiSU-Ultra/SukiSU-Ultra) 的分支，而 SukiSU-Ultra 本身基于 [KernelSU](https://github.com/tiann/KernelSU)。衷心感谢这两个上游项目。
:::

## 特性

1. 基于内核的 su 和权限管理。
2. 基于 Magic Mount 的模块系统。
3. App Profile: 把 Root 权限关进笼子里。
4. 支持 non-GKI 与 GKI 1.0。
5. KPM 支持
6. 可调整管理器外观，可自定义 susfs 配置。
